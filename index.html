squt (it's like SQL, but cUTe)

squt is a Perl and PHP/JS library aiming at graphically representing (My)SQL queries in a graph form.

squt uses :

  • Philip Stoev's MyParse to parse MySQL queries
  • Mike Bostock's d3 to handle the graph representation.
  • Marijn Haverbeke's CodeMirror for the in-browser code editor
  • Kevin Lindsey's js-intersections for geometrical intersection calculations.
  • jQuery's qunit and qunit-reporter-junit along with PrettyCode's Object.identical.js for testing purposes

All of them are integrated into squt as submodules : no need to install them manually.
